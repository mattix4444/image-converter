🔄 File Converter (Image · Text · Video)

A browser-based file converter available as a live website with a custom domain, supporting images, text, and video with no server required.

🌐 Live Website

👉 Visit the site here:
https://your-simple-converter.vercel.app/

✨ Features

🖼️ Image conversion: JPG, PNG, WEBP, GIF, BMP, SVG
📄 Text conversion: TXT, MD, HTML, CSV, JSON, XML, YAML, LOG
🎬 Video conversion: WEBM and MP4 (re-wrap)
📥 Automatic download after conversion
🖱️ Drag & drop support
📊 Progress bar
⚡ Fully client-side (no backend)
🎯 Tab-based interface
